# Spécifications GenEngine

Ce dossier décrit les intentions, invariants et décisions du projet.

- Le code et l'OpenAPI généré font autorité pour les contrats exécutables.
- Les ADR font autorité pour les décisions structurantes.
- Les tests et golden files fixent les comportements de compatibilité.
- Les migrations décrivent les transformations supportées.
